# GeoSpan - Interactive Matrix Editor & Linear Algebra Tool
一个交互式矩阵工具，用于线性代数中的行变换、RREF 化、零空间计算等操作。
基于 Python、PyQt5、SymPy 和 NumPy 构建。

![screenshot](assets/v1.20.png)

## 使用简介
- 单元格可直接输入浮点数或整数
- 空白单元格默认填0
- 支持创建与调整矩阵尺寸
- 支持加减乘除交换等行操作
- 支持一键化简RREF
- 支持一键生成核空间
- 支持一键清零矩阵
- 支持一键用1-9的随机整数填满矩阵
- 支持回退和重做所有步骤
- 可视化实现的行操作
- 可视化3D柱状图显示矩阵

## 行操作详细说明
Target Row [_A_] x [_p_] / [_q_] + [_k_] x Row [_B_]

### 行操作逻辑：
    [Target Row A] ← [(p / q) × Row A] + [k × Row B]

### 参数说明：
    A（目标行）      必填。指定要进行操作的行号。
    p（乘数）        可选。默认为 1。
    q（除数）        可选。默认为 1。
    k（第二行系数）  可选。默认为 1。
    B（第二行）      可选。指定参与加权的另一行行号。

### 操作逻辑：
    - 若仅填写 A、p、q，则执行缩放：Row A ← (p/q) × Row A
    - 若同时填写 k 和 B，则执行加权：Row A ← (p/q) × Row A + k × Row B
    - 所有参数支持浮点数输入，未填写的数值将使用默认值
